Class InitParameters

Struct containing all parameters passed to the SDK when initializing the ZED. These parameters will be fixed for the whole execution life time of the camera. For more details, see the InitParameters class in the SDK API documentation: https://www.stereolabs.com/developers/documentation/API/v2.5.1/structsl_1_1InitParameters.html
